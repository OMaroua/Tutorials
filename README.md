# Technical Tutorials

## Overview

This repository contains comprehensive, in-depth tutorials on deep learning, computer vision, and artificial intelligence topics. Each tutorial includes theoretical explanations, working code implementations, and visualizations.

## Available Tutorials

### 01 - Variational Autoencoder (VAE)

**Understanding Latent Space Through 2D and 3D Experiments**

A complete guide to Variational Autoencoders covering:
- VAE architecture and mathematical foundations
- 2D latent space visualization and clustering
- 3D latent space with increased capacity
- Non-isotropic Gaussian priors with covariance

**Difficulty:** Intermediate  
**Duration:** 2-3 hours  
**Topics:** Generative Models, Deep Learning, TensorFlow

## Tutorial Format

Each tutorial follows a consistent structure:

1. **Theoretical Foundation**
   - Mathematical explanations
   - Intuitive understanding
   - Key concepts and terminology

2. **Implementation**
   - Step-by-step code
   - Architecture details
   - Training procedures

3. **Experiments & Results**
   - Reproducible experiments
   - Visualizations and analysis
   - Performance metrics

4. **Extensions**
   - Customization options
   - Advanced topics
   - Further reading
